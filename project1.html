Dota 2 Chat Web Application

Description

Chat Web application that features account registration and chat room creation using NodeJS Express on the server side, RESTful API and sockets for client-server communication, HTML5+BOOTSTRAP+JQuery for the client side.

Visit Page

Heroku Git URL: https://git.heroku.com/jediquinmidtermproject.git

Reflection

When it comes to presenting something that is visually appealing, this is my best work. This is the first major web application project that I’ve done where I set up a server and used RESTful API, web sockets, and bootstrap. I have applied all the things that I’ve learned from the web development courses I took at BCIT and the things that I taught myself which I think is very useful in web development.

The project was not that hard, but it would test how well you know how RESTful API and sockets work. I first implemented the functionalities and created a simple page. As I slowly develop this project, the vision of how I wanted it to look like became clearer. Page by page, element by element, I started structing things. I had minor problems with the layout because bootstrap was not really taught to us, but a little Googling and research fixed the problem.

As I was getting close to completion, I've added more features to it than what our instructor required. For me, this was not only just a course project to satisfy requirements, this became a project I can show to employers.

I had so much fun in making this project. I’m very happy on how it turned out and very proud of this work.
